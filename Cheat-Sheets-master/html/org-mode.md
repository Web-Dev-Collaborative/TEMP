Syntax
------

{: .-three-column}

### Headings

    * Welcome to Org mode

      Lines starting with * are headings.
      These lines without are notes.

    ** Sub-heading

       Two stars mark a 2nd-level subheading (h2).

### Lists

    * Lists

    To buy:
    1. Milk
    2. Eggs
       - Organic
    3. Cheese
       + Parmesan
       + Mozarella

### Inline styles

    *bold*
    /italic/
    _underline_
    =code=
    ~verbatim~
    +strike-through+

### To do

    * TODO buy airplane

Cycle by using `S-LEFT` / `S-RIGHT`. List all TODO’s via `C-c C-v`.
