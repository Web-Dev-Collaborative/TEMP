### Modifiers

The following CSS classes affect the **colour**.

    .is-primary
    .is-link
    .is-info
    .is-success
    .is-warning
    .is-danger

The following classes modify the **size**.

    .is-small
    .is-medium
    .is-large

    .is-outlined
    .is-loading
