### Input

     <input ...
       disabled
       required
       checked

       autofocus

       autocomplete='off'        <!-- autocomplete -->
       autocompletetype='cc-exp'
       autocapitalize='off'      <!-- for mobiles -->
       pattern='\d*'             <!-- force numeric input in iOS -->

### Input types

#### Text

-   email
-   hidden
-   **password**
-   tel
-   **text**
-   search {: .-four-column}

#### Time

-   date
-   time {: .-four-column}

#### Time (not widely supported)

-   month
-   week
-   datetime
-   datetime-local {: .-four-column}

#### Etc

-   **file**
-   **radio**
-   **checkbox** {: .-four-column}

#### Buttons

-   button
-   reset
-   submit
-   image {: .-four-column}

#### Numeric

-   number
-   range {: .-four-column}
