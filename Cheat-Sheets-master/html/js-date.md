Date
----

{: .-left-reference}

### Constructor

    // Now
    new Date()

    // ms since epoch
    new Date(1419785527580)

    // Date format
    new Date("May 17, 1995 03:24:00")

    // ISO date format
    new Date("2013-03-01T01:10:00")

    new Date(2014, 2, 1, 13, 0, 59, 0)

### Constructor

`new Date(` | `2014,` | `2,` |`1,` | `13,` | `0,` | `59,` |`0)` |  
Date | Year | Month | Day | Hour | Min | Sec | Milli |

{: .-css-breakdown}

Months are zero-indexed (eg, January is `0`).
