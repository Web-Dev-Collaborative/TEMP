### Scripts

    "scripts": {
      "start": "node ./bin/xxx",       /* npm start */
      "test": "vows --spec --isolate", /* npm test */
      "postinstall": "...",
      "prepublish": "grunt build",     /* after 'npm install' and before 'npm 
                                          publish' */
    }

### Misc

    "private": true,
    "preferGlobal": true

### Config

    {
      "config": {
        "foobar": "hello"
      },
      "scripts": {
        "run": "echo $npm_package_config_foobar"
      }
    }

Keys in `config` are exposed as env vars to scripts.
