MediaElementAudioSourceNode.mediaElement
========================================

<span class="seoSummary">The [`MediaElementAudioSourceNode`](../MediaElementAudioSourceNode.html) interface's read-only `mediaElement` property indicates the [`HTMLMediaElement`](../HTMLMediaElement.html) that contains the audio track from which the node is receiving audio.</span> This stream was specified when the node was first created, either using the [`MediaElementAudioSourceNode()`](MediaElementAudioSourceNode.html "MediaElementAudioSourceNode()") constructor or the [`AudioContext.createMediaElementSource()`](../AudioContext/createMediaElementSource.html) method.

[Syntax](#syntax "Permalink to Syntax")
---------------------------------------

    audioSourceElement = mediaElementAudioSourceNode.mediaElement;

### [Value](#value "Permalink to Value")

An [`HTMLMediaElement`](../HTMLMediaElement.html) representing the element which contains the source of audio for the node.

[Examples](#examples "Permalink to Examples")
---------------------------------------------

    const audioCtx = new window.AudioContext();
    const audioElem = document.querySelector('audio');

    let options = {
      mediaElement: audioElem
    }

    let source = new MediaElementAudioSourceNode(audioCtx, options);
    console.log(source.mediaElement);
