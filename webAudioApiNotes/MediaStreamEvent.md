MediaStreamEvent
================

#### Deprecated

This feature is no longer recommended. Though some browsers might still support it, it may have already been removed from the relevant web standards, may be in the process of being dropped, or may only be kept for compatibility purposes. Avoid using it, and update existing code if possible; see the [compatibility table](#browser_compatibility) at the bottom of this page to guide your decision. Be aware that this feature may cease to work at any time.

The **`MediaStreamEvent`** interface represents events that occurs in relation to a [`MediaStream`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream). Two events of this type can be thrown: `addstream` and `removestream`.

[Properties](#properties "Permalink to Properties")
---------------------------------------------------

*A [`MediaStreamEvent`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamEvent) being an [`Event`](https://developer.mozilla.org/en-US/docs/Web/API/Event), this event also implements these properties*.

[`MediaStreamEvent.stream`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamEvent/stream) <span class="badge inline readonly" title="This value may not be changed.">Read only </span>  
Contains the [`MediaStream`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream) containing the stream associated with the event.

[Constructors](#constructors "Permalink to Constructors")
---------------------------------------------------------

[`MediaStreamEvent()`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamEvent/MediaStreamEvent "MediaStreamEvent()")  
Returns a new `MediaStreamEvent`. It takes two parameters, the first being a [`DOMString`](https://developer.mozilla.org/en-US/docs/Web/API/DOMString) representing the type of the event; the second a dictionary containing the [`MediaStream`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStream) it refers to.

[Methods](#methods "Permalink to Methods")
------------------------------------------

A [`MediaStreamEvent`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamEvent) being an [`Event`](https://developer.mozilla.org/en-US/docs/Web/API/Event), this event also implements these properties.¬†There is no specific [`MediaStreamEvent`](https://developer.mozilla.org/en-US/docs/Web/API/MediaStreamEvent) method.

    };
